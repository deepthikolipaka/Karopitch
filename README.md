# Karo Pitch — Demo Landing Page

A demo homepage designed for **Karo Pitch** by KaroStartup as part of an intern assignment.

Karo Pitch is a platform where early-stage founders from across India — especially from Tier-2, Tier-3 cities and Bharat — can pitch their businesses directly to investors and raise funding.

---

## Live Demo

[View Live Site]((https://github.com/deepthikolipaka/Karopitch/))

---

## Built With

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome 6.5 (icons)
- Google Fonts — Syne + DM Sans

---

## Sections Covered

- Hero — headline, subtext, CTA buttons, stats
- About Karo Pitch — mission and vision
- How It Works — 4-step process
- Who Can Apply — startup categories
- Investors — placeholder investor section
- Featured Startups — dummy startup cards
- About KaroStartup — platform background and stats
- Final CTA — Apply Now and Partner With Us

---

## Features

- Fully responsive and mobile friendly
- Smooth scroll animations
- Hamburger menu for mobile
- Scroll reveal on sections
- Staggered card animations
- Fixed navbar with scroll shadow
